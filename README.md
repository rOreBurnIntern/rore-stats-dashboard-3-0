## About

A lightweight, single-page stats dashboard for the rORE mining protocol. Displays live token prices, Motherlode TVL, and round analytics by fetching data from a GitHub Gist data source.

## Why Archived?

This v1 approach had limitations (no server-side processing, fragile Gist pipeline, no error monitoring). Development moved to a full Next.js stack in v2.
